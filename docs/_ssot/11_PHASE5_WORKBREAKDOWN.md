# Phase 5 상세 실행 계획(SSOT)

Phase 5의 목표는 **탐색/검색/추천 + 기획사 운영 기능**을 체계화하는 것이다.

## 범위
- 공개 채널 탐색/검색
- 기획사 북마크/평가 메모
- 기본 추천/정렬 기준

## 비범위
- 고급 ML 추천
- 광고/프로모션
- 결제 상품 확장

## 작업 스트림

### 1) 탐색/검색
#### 목표
- 기획사가 채널을 효율적으로 탐색 가능

#### 작업 항목
- 검색 조건(국가/카테고리/정렬) 확정
- 기본 검색 API

#### 완료 조건
- 필터/정렬 기반 탐색 가능

---

### 2) 기획사 운영
#### 목표
- 기획사 내부 평가/관리 기반 구축

#### 작업 항목
- 북마크/메모 모델 및 API
- 담당자별 메모 분리 정책

#### 완료 조건
- 기획사 계정으로 운영 기능 사용 가능

---

## 현재 구현(최소)
- 테이블: `agency_bookmarks`
- API:
  - `GET /api/v1/channels/search`
  - `GET /api/v1/agency/bookmarks`
  - `POST /api/v1/agency/bookmarks`
  - `DELETE /api/v1/agency/bookmarks/{applicantId}`
