# 제품 비전(SSOT)

이 문서는 “Global Audition Platform”의 제품 비전과 핵심 사용자 흐름을 **단일 진실 원천(SSOT)** 으로 유지한다.

## 1) 문제/가치
- **지원자**: 포트폴리오(영상) 기반으로 기획사에게 발견되고 싶다
- **기획사**: 오디션 공고/지원자 풀을 효율적으로 평가·선발하고 싶다
- **본사/운영**: 서비스 안전성, 권한/정책 일관성, 글로벌 확장을 유지하고 싶다

## 2) 핵심 기능(현재/가까운 목표)
- 오디션 공고 생성/관리
- 지원자 지원(유튜브 영상 기반) 및 심사 단계(1차~3차, 가변) 처리
- 지원자 개인 채널(유튜브 유사): 영상/프로필/콘텐츠를 지속적으로 운영

## 3) 확장 기능(로드맵 반영)
- 트레이너 피드백(유료 가능성 포함)
- 칼럼/콘텐츠 카테고리(칼럼니스트 역할)
- 오디션 분야 확장: 가수/댄서 → 모델/연기 등

## 4) 글로벌 전략
- UI i18n(키 기반) + 도메인 콘텐츠 번역(원문+번역본)
- 번역 시스템은 provider 추상화로 교체 가능하게(벤더 종속 최소화)

## 5) 결제
- Stripe 기반 결제/정산/환불/웹훅 처리

## 6) 핵심 사용자 흐름(요약)
### 지원자
1. 가입/인증 → 프로필/채널 세팅
2. 오디션 탐색 → 영상(유튜브)로 지원
3. 단계별 결과 확인/추가 제출
4. 채널 콘텐츠 운영(지속 업데이트)

### 기획사
1. 가입/승인(정책) → 조직/담당자 관리
2. 오디션 생성 → 지원자 모집
3. 지원자 평가/단계 이동/선발
4. 채널/콘텐츠 탐색으로 인재 발굴
