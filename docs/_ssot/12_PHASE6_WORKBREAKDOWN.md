# Phase 6 상세 실행 계획(SSOT)

Phase 6의 목표는 **운영/안정성/성장 지표**를 강화하는 것이다.

## 범위
- 운영 대시보드/통계
- 감사 로그/이벤트 추적
- 알림/모니터링

## 비범위
- 대규모 데이터 웨어하우스
- 실시간 스트리밍 분석

## 작업 스트림

### 1) 운영 지표
#### 목표
- KPI 기반 운영 가능

#### 작업 항목
- 오디션/지원/전환 지표 정의
- 관리자용 통계 API

#### 완료 조건
- 기본 KPI 대시보드 구성 가능

---

### 2) 안정성/운영
#### 목표
- 장애 대응/감사 가능성 확보

#### 작업 항목
- 감사 로그 설계
- 주요 이벤트 알림/모니터링 연동

#### 완료 조건
- 장애/이상 감지 시 대응 흐름 확보

---

## 현재 구현(최소)
- 테이블: `audit_logs`
- API:
  - `POST /api/v1/admin/audit-logs`
  - `GET /api/v1/admin/audit-logs`
  - `GET /api/v1/auditions/admin/stats`
