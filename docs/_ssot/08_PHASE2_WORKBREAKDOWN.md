# Phase 2 상세 실행 계획(SSOT)

Phase 2의 목표는 **개인 채널(발견/탐색)** 기반을 확립하는 것이다.

## 범위
- 채널 프로필(소개/배너/링크/공개 설정)
- 채널 공개/비공개 정책
- 채널 영상 목록 조회(미디어 서비스 연동)

## 비범위
- 트레이너 피드백/칼럼
- 결제/정산
- 번역 파이프라인

## 작업 스트림

### 1) 채널 프로필
#### 목표
- 지원자 채널 프로필을 저장/조회/수정 가능

#### 작업 항목
- 채널 공개 범위(`PUBLIC/UNLISTED/PRIVATE`) 추가
- 채널 프로필 조회/수정 API

#### 완료 조건
- 공개 채널 조회 가능
- 본인만 수정 가능

---

### 2) 채널 콘텐츠 목록
#### 목표
- 채널 영상 목록 조회

#### 작업 항목
- 미디어 서비스의 `userId` 기반 영상 조회 연동
- 게이트웨이를 통한 호출 경로 정리

#### 완료 조건
- 채널 화면에서 영상 목록이 표시됨

---

### 3) 탐색/검색 기본
#### 목표
- 기획사가 채널을 탐색할 수 있는 최소 기능

#### 작업 항목
- 공개 채널 필터(카테고리/정렬) 정의
- 초기 검색 조건(국가/카테고리/정렬 기준) 정의

#### 완료 조건
- 최소 탐색 UX가 동작
