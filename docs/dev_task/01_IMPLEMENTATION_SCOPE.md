SCOPE:
- 영상 업로드 API 확정 (파일 업로드 or URL 중 기존 방식 고정)
- 영상 메타데이터: title, description, tags(작사/작곡/커버/장르), visibility
- 내 채널 영상 리스트 조회 (최신순)
- 오디션 지원 시 영상 선택/연결

RULES:
- 트랜스코딩/썸네일 자동 생성 금지
- 저작권 검증 금지
- 추천/랭킹 연동 금지

DEFINITION OF DONE:
- 소유자만 수정/삭제 가능
- PRIVATE 영상은 소유자/ADMIN만 조회 가능
- 오디션 제출 영상 정상 연결 (E2E 테스트)
